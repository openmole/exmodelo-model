## Build console

```sh
> cd scala
> sbt
> project console
sbt> run
```

## Build Zombieland

```sh
> cd scala
> sbt
sbt> project gui
sbt> buildGUI
```

Then browse:
```sh
scala/zombieland/target/zombies.html
```
